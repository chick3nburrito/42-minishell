Rebuild the core mechanisms of a shell from scratch


Minishell is a system programming project from the 42 curriculum,
focused on building a simplified Unix shell from scratch.
It recreates core behaviors of Bash while teaching low-level concepts such as process control, signals, file descriptors, and system calls.

✨ Features

Command execution with arguments and environment variables

Pipes (|) for chaining commands

Redirections (<, >, >>, <<)

Built-in commands: cd, echo, pwd, env, export, unset, exit

Environment variable expansion ($VAR)

Signal handling (Ctrl-C, Ctrl-D, Ctrl-\)

Error output similar to Bash
